# uoa-el331-team-tomodachi
Authorship Analysis using Python

## 実行方法
### server

```sh
cd server
python3 -m pip install -r requirements.txt
python3 main.py
```

### front

```sh
cd front
npm install
npm run dev
```

### Docker
```sh
docker compose up
```

### For muizu
. env/bin/activate
